
Encontrar la potencia de un número elevado a la 2:
1. Definir una variable para almacenar el número base.
2. Solicitar al usuario que ingrese el número base.
3. Elevar el número base a la potencia de 2 utilizando el operador de potencia.
4. Mostrar el resultado de la potencia al usuario.

1. Definir una variable para almacenar el número a calcular el factorial.
2. Solicitar al usuario que ingrese el número.
3. Inicializar una variable para almacenar el factorial (inicializarla en 1).
4. Recorrer desde 1 hasta el número ingresado, multiplicando el factorial por cada número en el rango.
5. Mostrar el resultado del factorial al usuario.
Contar letras de un nombre y mostrar un mensaje:
1. Definir una variable para almacenar el nombre.
2. Solicitar al usuario que ingrese su nombre.
3. Contar la cantidad de letras en el nombre usando la función de longitud de cadena.
4. Si la cantidad de letras es menor a 10, mostrar el mensaje "Hola, tu nombre es muy corto".
5. Si la cantidad de letras es mayor o igual a 10, mostrar un mensaje genérico como "Hola, [nombre del usuario]".
Tablas de multiplicar de un número:
1. Definir una variable para almacenar el número del que se quieren obtener las tablas de multiplicar.
2. Solicitar al usuario que ingrese el número.
3. Recorrer desde el 1 hasta el 10.
4. Dentro del ciclo, multiplicar el número por el valor actual del ciclo.
5. Mostrar el resultado de la multiplicación.
6. Repetir el ciclo para cada número del 1 al 10.
Comprobar si un número es primo y mostrar divisores:
1. Definir una variable para almacenar el número a evaluar.
2. Solicitar al usuario que ingrese el número.
3. Inicializar una variable para contar divisores (inicializarla en 0).
4. Recorrer desde 2 hasta la raíz cuadrada del número ingresado.
5. Si el número es divisible por el valor actual del ciclo, incrementar el contador de divisores y agregar el número a una lista de divisores.
6. Después del ciclo, verificar si el contador de divisores es igual a 1.
7. Si el contador es igual a 1, el número es primo y se muestra un mensaje indicando que es primo.
8. Si el contador es mayor a 1, el número no es primo y se muestra un mensaje indicando que no es primo y se imprime la lista de divisores.
